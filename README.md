# Лабиринт сокровищ

Текстовая приключенческая игра на Python, где игрок исследует лабиринт, собирает предметы, решает загадки и ищет сокровища.

## Геймплей
![](demo.gif)

## Описание

"Лабиринт сокровищ" — это консольная игра, в которой вы играете роль отважного исследователя в древнем лабиринте. Ваша цель — найти главное сокровище, преодолевая различные препятствия, решая головоломки и собирая полезные предметы.

## Установка

Убедитесь, что у вас установлен Python 3.11 или выше и poetry.

```bash
# Установка зависимостей
make install
# или
poetry install
```

## Запуск

```bash
# Запуск игры
make project
# или
poetry run project
```

## Доступные команды в игре

- `go <направление>` или просто `<направление>` - перейти в указанном направлении (north, south, east, west)
- `look` - осмотреть текущую комнату
- `take <предмет>` - поднять предмет
- `use <предмет>` - использовать предмет из инвентаря
- `inventory` - показать инвентарь
- `solve` - попытаться решить загадку в комнате
- `help` - показать список команд
- `quit` или `exit` - выйти из игры

## Разработка

```bash
# Проверка стиля кода
make lint

# Сборка пакета
make build

# Публикация (dry-run)
make publish
```
